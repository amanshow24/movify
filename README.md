
### 2. Get a TMDB API Key

* Visit [TMDB](https://www.themoviedb.org/)
* Create an account and generate an API key from your dashboard.

### 3. Add Your API Key

Create a `config.js` file in the root or `js` folder (wherever you use the API) and add:

